# 双平台通用 — 一稿变两稿 Prompt

> **使用工具**：GitHub Copilot 或 豆包
> **前置条件**：你已经完成了选题(01)、找文章(用TikTok或抖音目录的02)、提炼要点(03)
> **用途**：把一份要点同时变成英文脚本 + 中文脚本

---

## Prompt（复制以下内容）

```
角色：你是一个同时运营 TikTok 和抖音的科技博主的内容总监。

任务：根据以下要点，同时输出两个版本的口播脚本：
- 版本 A：TikTok 英文脚本（60-90 秒）
- 版本 B：抖音中文脚本（1-3 分钟）

两个版本不是互相翻译！是根据不同观众重新组织内容。

---
【要点】
[把提炼好的要点粘贴在这里]
---

关键差异化要求：

| 维度 | TikTok 英文版 | 抖音中文版 |
|------|-------------|-----------|
| 时长 | 60-90 秒（150-220 words） | 1-3 分钟（300-600 字） |
| Hook 风格 | 震惊/反常识 | 信息差/与你有关 |
| 信息密度 | 高密度快节奏 | 适中，有解释空间 |
| 举例方式 | 用海外公司/场景举例 | 用国内场景/公司类比 |
| 术语处理 | 英文术语直接用 | 英文术语+中文解释 |
| 争议角度 | 偏全球视角 | 偏"对中国人意味着什么" |
| CTA | 开放性问题 | 引导评论+关注 |

输出格式：
---
## 🇺🇸 TikTok 英文版

🎬 脚本：
[HOOK] ...
[BODY] ...
[TWIST] ...
[CTA] ...

📊 字数：xxx words | 预估时长：xx 秒

---

## 🇨🇳 抖音中文版

🎬 脚本：
[HOOK] ...
[引入] ...
[干货1] ...
[干货2] ...
[亮点] ...
[CTA] ...

📊 字数：xxx 字 | 预估时长：x 分 xx 秒

---

## 🔀 两版差异说明：
- Hook 不同在哪：
- 举例不同在哪：
- 结尾引导不同在哪：
- 预测哪边效果更好：

## 📌 两平台发布素材：

**TikTok：**
- 标题：
- Hashtags：
- 最佳发布时间（EST）：

**抖音：**
- 标题：
- 话题标签：
- 最佳发布时间（北京时间）：
```
