# TikTok 提炼要点 Prompt

> **使用工具**：GitHub Copilot（在 VS Code 里用） 或 ChatGPT
> **前置条件**：把 Step 2 找到的文章内容复制过来

---

## Prompt（复制以下内容，把文章内容粘贴到指定位置）

```
角色：你是一个跨境科技内容策划师，擅长把中文科技资讯提炼成海外观众能理解的叙事。

任务：根据以下文章素材，提炼出一个 60-90 秒 TikTok 口播视频的核心要点。

---
【素材开始】
[把 Step 2 搜集的所有文章要点粘贴在这里]
【素材结束】
---

提炼要求：
1. 用"海外观众视角"重新组织信息（他们不了解中国科技圈背景）
2. 找出 1 个最有冲击力的核心论点（hook）
3. 找出 2-3 个支撑论点的关键事实/数据
4. 找出 1 个能引发评论区讨论的争议观点
5. 去掉所有中国特有语境（不需要解释的背景知识）

输出格式：
---
🎯 核心论点（Hook）：
[一句话，要让人停下来看——"等等，这是真的？"]

📊 关键事实（按冲击力排序）：
1. [最震撼的数据/事实]
2. [第二个支撑点]
3. [第三个支撑点]

🔥 争议观点：
[一个能引发评论区辩论的角度]

❌ 删掉的信息（不适合海外观众的部分）：
- [列出你删掉了什么，为什么]

🧱 建议叙事结构：
Hook → 事实1 → 事实2 → 转折/对比 → 争议观点 → CTA
---
```
