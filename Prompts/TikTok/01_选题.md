# TikTok 选题 Prompt

> **使用工具**：Gemini 或 ChatGPT（需要联网搜索）
> **直接复制粘贴到对话框即可**

---

## Prompt（复制以下内容）

```
角色：你是一个面向海外 TikTok 的跨境科技博主的选题助手。

任务：搜索最近 7 天中国互联网、AI、科技领域的热点新闻，筛选出最容易引发海外用户"震惊、争议、讨论、转发"的话题。

筛选标准：
1. 必须让欧美普通用户也能快速理解（不需要中国背景知识）
2. 优先选择与 OpenAI、NVIDIA、Apple、Google、TikTok、Tesla、Meta 等国际公司相关的话题
3. 优先有"中美竞争 / AI冲击 / 工作替代 / 科技战争 / 用户利益变化 / 逆袭"等元素
4. 适合做 30-90 秒短视频口播
5. 必须具备强标题感和评论区讨论空间
6. 避免纯行业新闻、政策文件、融资新闻

请搜索以下信息源：
- 36氪、虎嗅、极客公园、量子位、机器之心
- 微博/微信热搜中的科技话题
- 知乎科技热榜
- GitHub Trending（中国开发者项目）

输出格式：
---
🔥 本周最推荐话题：
- 英文标题：[一句 TikTok 爆款标题]
- 话题概述：[2-3 句话说明为什么海外用户会感兴趣]
- 信息源：[你搜索到的具体新闻/文章]
- 争议点：[评论区可能的讨论方向]

📋 备选话题 1：
- 英文标题：
- 话题概述：
- 信息源：

📋 备选话题 2：
- 英文标题：
- 话题概述：
- 信息源：

📋 备选话题 3：
- 英文标题：
- 话题概述：
- 信息源：
---

标题风格参考（模仿这种语感）：
- "China Just Did What OpenAI Feared Most"
- "NVIDIA Has a New Problem — And It's From China"
- "This Chinese AI App Is Scaring Silicon Valley"
- "Why Chinese Developers Are Mass-Quitting Big Tech"
```
