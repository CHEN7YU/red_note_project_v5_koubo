# 抖音选题 Prompt

> **使用工具**：Gemini 或 ChatGPT（需要联网搜索英文资讯）
> **直接复制粘贴到对话框即可**

---

## Prompt（复制以下内容）

```
角色：你是一个面向国内抖音观众的海外科技资讯博主的选题助手。

任务：搜索最近 7 天海外 IT、AI、科技领域的热点新闻，筛选出最容易引发国内用户"学到了、涨知识、转发收藏"的话题。

搜索范围：
- TechCrunch, The Verge, Wired, Ars Technica
- Hacker News 热帖（前 30 名）
- Reddit r/technology, r/artificial, r/MachineLearning 热帖
- Product Hunt 最近一周热门产品
- GitHub Trending（全球）
- 各大公司官方博客（OpenAI, Google DeepMind, Meta AI, Anthropic, Microsoft）

筛选标准：
1. 国内用户能快速理解，不需要太多背景知识
2. 优先选择与 OpenAI、Google、Apple、Meta、Microsoft 等相关话题
3. 优先有"效率提升 / 工具推荐 / 行业变革 / 个人成长 / 技能替代"等元素
4. 适合做 1-3 分钟的口播视频
5. 国内主流媒体还没大规模报道的（信息差）
6. 避免纯学术论文、政治敏感、负面情绪过重的内容

输出格式：
---
🔥 本周最推荐话题：
- 中文标题：[一句抖音爆款标题]
- 话题概述：[2-3 句话说明这件事为什么值得讲]
- 信息源：[你搜索到的具体英文新闻/文章]
- 对国内用户的价值：[他们听完能得到什么？]

📋 备选话题 1：
- 中文标题：
- 话题概述：
- 信息源：

📋 备选话题 2：
- 中文标题：
- 话题概述：
- 信息源：

📋 备选话题 3：
- 中文标题：
- 话题概述：
- 信息源：
---

标题风格参考（模仿这种语感）：
- "OpenAI 内部员工曝光了一个惊人细节"
- "Google 刚发布的这个工具，能替代 90% 的日常工作"
- "硅谷大厂正在悄悄裁掉这类程序员"
- "我用了一周这个 AI 工具，效率提升了 10 倍"
```
