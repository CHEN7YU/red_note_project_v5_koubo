## 每周 5 步流程（每步用什么工具）

### Step 1️⃣ 选题（5 分钟）
- **工具**：Gemini（免费版自带 Google 搜索） 或 ChatGPT（免费版有联网）
- **Prompt 文件**：
  - TikTok → `Prompts/TikTok/01_选题.md`
  - 抖音 → `Prompts/抖音/01_选题.md`
- **为什么用这个工具**：需要实时联网搜索最近一周的热点，Gemini 和 ChatGPT 免费版都能联网

### Step 2️⃣ 找文章（10 分钟）
- **工具**：Gemini（搜索能力最强） 或 M365 Copilot（Bing 搜索）
- **Prompt 文件**：
  - TikTok → `Prompts/TikTok/02_找文章.md`
  - 抖音 → `Prompts/抖音/02_找文章.md`
- **为什么用这个工具**：需要找到具体文章链接和内容，Gemini 的 Google 搜索最精准

### Step 3️⃣ 提炼要点（10 分钟）
- **工具**：GitHub Copilot（长上下文能力强） 或 ChatGPT
- **Prompt 文件**：
  - TikTok → `Prompts/TikTok/03_提炼要点.md`
  - 抖音 → `Prompts/抖音/03_提炼要点.md`
- **操作方式**：把 Step 2 搜到的文章内容粘贴进去，让 AI 提炼
- **为什么用这个工具**：不需要联网，需要强分析能力，Copilot/ChatGPT 最擅长

### Step 4️⃣ 写口播脚本（15 分钟）
- **工具**：
  - TikTok 英文脚本 → GitHub Copilot 或 ChatGPT
  - 抖音中文脚本 → 豆包（中文表达最自然）或 GitHub Copilot
- **Prompt 文件**：
  - TikTok → `Prompts/TikTok/04_写脚本.md`
  - 抖音 → `Prompts/抖音/04_写脚本.md`
- **为什么用这个工具**：豆包的中文口语化能力强；Copilot/ChatGPT 英文输出质量高

### Step 5️⃣ 生成标题 + 封面文案 + 标签（5 分钟）
- **工具**：同 Step 4 的工具
- **Prompt 文件**：
  - TikTok → `Prompts/TikTok/05_标题封面标签.md`
  - 抖音 → `Prompts/抖音/05_标题封面标签.md`

## 建议节奏

### 模式 A：每周 1 话题 × 2 平台（推荐起步）
- **周一**：用 `双平台通用/01_选题.md` 选一个中外通吃的话题
- **周二**：找文章 + 提炼要点（用 TikTok 或 抖音 的 02-03）
- **周三**：用 `双平台通用/02_一稿变两稿.md` 一次出两个脚本
- **周四**：录制（先录一个语言，再录另一个）
- **周五-周日**：分别发布 + 互动

### 模式 B：每周 2 话题（进阶）
- 1 个双平台话题（出 2 条）+ 1 个平台专属话题（出 1 条）= 每周 3 条

稳定后可以提速到每周 4-6 条。
