# TikTok 找文章 Prompt

> **使用工具**：Gemini（Google 搜索最强）或 M365 Copilot
> **前置条件**：先完成 Step 1 选题，拿到确定的话题

---

## Prompt（复制以下内容，把 [话题] 替换成你选定的话题）

```
任务：围绕以下话题，帮我找到 3-5 篇高质量的中文原始报道/文章，并提取关键内容。

话题：[把 Step 1 选定的话题粘贴在这里]

搜索要求：
1. 搜索 36氪、虎嗅、极客公园、量子位、机器之心、InfoQ、少数派 等科技媒体
2. 搜索知乎相关高赞回答
3. 搜索微博/微信公众号相关讨论
4. 如果有 GitHub 项目相关，搜索 GitHub README 和 Star 数据
5. 时间范围：最近 14 天内

对每篇文章，输出：
---
📄 文章 1：
- 标题：
- 来源/链接：
- 发布日期：
- 核心信息（3-5 个要点）：
  1. 
  2. 
  3. 
- 数据/数字亮点：[任何能让视频更有说服力的数据]
- 原文金句：[值得直接引用的原话]

📄 文章 2：
...（同上格式）

📄 文章 3：
...（同上格式）
---

最后，综合所有文章给出：
🧠 综合信息密度评估：
- 这些文章是否足够支撑一个 60-90 秒的口播？
- 还缺什么角度？需要补充搜索什么？
```
